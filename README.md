# Jokes API Practice

-   Restful API
-   Django (DRF)
-   Dockerization

---

**Commands :**

```
pip install poetry
poetry install -n --no-root
poetry run python manage.py runserver
```

---

**Http resquest:**

-   All Jokes (GET, POST)
    `http://localhost:8000/jokes/`
-   Single Joke (GET, PUT PATCH, DELETE)
    `http://localhost:8000/jokes/<id>`
